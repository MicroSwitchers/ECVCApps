The Workshop

Welcome to my digital workshop, a collection of my ongoing projects dedicated to exploring technology in blind and low-vision early intervention. This space serves as a workbench for ideas, featuring a mix of 3D-printed tools, vibe-coded web applications, and microcontroller-based devices.

Some of these projects are polished and may be ready for everyday use, while others are active experiments and proofs-of-concept. The goal is to innovate and share possibilities, so please feel free to explore, adapt, and build upon these projects.

Please see each project's GitHub page for licensing information. Most are released under permissive open-source licenses, such as MIT, to encourage building off these ideas.

Gaze Tracking

Tools designed for use with gaze tracking hardware

Gaze Tracking
V1

GazeCanvas v1

Version 1 of GazeCanvas is a minimal image canvas designed to be efficiently manage visual targets like images, video and text for use with a gaze tracking device overlay. Simple movement animations can be applied to the targets and triggered from the side bar. This requires a gaze tracking device with its own overlay software.

Use GazeCanvas v1
Gaze Tracking
V2

GazeCanvas v2

Version 2 of GazeCanvas is an image canvas designed to be efficiently manage visual targets like images, video, shapes and text for use with a gaze tracking device overlay. Animation can be applied to the targets and triggered by pressing the numbers on a standard keyboard. This requires a gaze tracking device with its own overlay software.

Use GazeCanvas v2

Mechanical Braille Writer Simulator

Practice using a mechanical Braille writer with this interactive simulator. It emulates the writer's features and limitations to help build muscle memory, while conserving paper. The app supports two input methods: use the sdf and jkl keys on a standard keyboard, or press the virtual keys directly on a tablet's screen. For the best experience using touch on a tablet, install the app to your device from your web browser's settings and use full-screen landscape mode.

Use the Mechanical Braille Writer Simulator
Braille
Keyboard

SixKey - Braille Practice Pad

A minimal Braille writing jot pad for practicing 6 key input with a standard computer keyboard using the sdf and jkl keys to produce braille cells. The app is intended to be small and efficient with minimal features or distractions. A dark mode is provided to reduce eye strain, a zoom feature can enlarge the cells, and an import and export feature allows the text to be saved and opened. It exports a standard Unicode (.txt) file, allowing you to open your work as Braille cells in any basic text editor.

Use the SixKey - Braille Practice Pad
Braille
Tablet Keyboard

BrailleFlex

Practice Braille with flexible key placement. This app works with a keyboard's `sdf` and `jkl` keys, or with adjustable on-screen touch keys that can be shifted for a more comfortable position on a tablet. It's a quick and basic tool for practicing 6 key input visually and not a full featured Braille app. For best results using the touch keys install the app through your browser and use landscape mode in full screen.

Use BrailleFlex

Concept

3D Printed Single Switch

This is a very early brainstorming project to look at how to design inexpensive adapted single switches that respond to very light activation force. I created it in Blender 3D and am providing a Blender file and an STL 3D print file for download. The switch is currently pinned on one side to make it a flap switch, but I'm researching ways to make it a light touch button switch. Ideas and feedback are appreciated.

About These Projects

Privacy Note: These apps process data locally. They do not send any information outside of your browser. None of the apps will ask you to input any identifying information or context. For example, age calculators will never allow the user to input anything other than an arbitrary date, disconnected from any context.
